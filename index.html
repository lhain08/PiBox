Welcome to the PiBox Site

This is the homepage of the PiBox, Install it onto your raspberry pi now by entering the following into lx terminal:

$ git clone https://github.com/slapshot135/PiBox

Want to Load Files Off of Flash Drives?

take any ordinary flash drive and rename it on your computer as PIDRIVE. Now insert it into your Pi and start up PiBox by entering:

$ cd PiBox
$ sudo python PiBox.py

any .py files on your Pi will automatically be moved into the PiBox folder and added as an option in the main menu
